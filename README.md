CodeAlpha File Organizer Automation

Project Overview

This project is a Python-based file organizer that automatically sorts files into different folders according to their file type.

Features

- Automatically creates folders
- Organizes image files (.jpg, .jpeg, .png)
- Organizes PDF files (.pdf)
- Organizes text files (.txt)
- Organizes video files (.mp4)
- Simple and user-friendly automation

Technologies Used

- Python
- os Module
- shutil Module

How It Works

1. Reads all files from the selected folder.
2. Creates folders for different file categories.
3. Detects file extensions.
4. Moves files into their respective folders automatically.

Example

Before:

photo.jpg
resume.pdf
notes.txt
movie.mp4

After:

Images/photo.jpg
PDFs/resume.pdf
TextFiles/notes.txt
Videos/movie.mp4
